# CDRPS Roadmap

This roadmap outlines the planned delivery phases for the Construction Delay Risk Prediction System.

## Completed

- Data cleaning and preparation pipeline
- PCA dimensionality reduction
- KMeans clustering
- Delay Risk Index computation
- Streamlit dashboard foundation

## Next

- Train a predictive model for delay risk classification or scoring
- Persist trained artifacts for reuse in the dashboard
- Add a prediction page to the Streamlit app
- Expand visual analytics for cluster comparison

## Later

- Add export options for cleaned data and risk summaries
- Improve dashboard branding and layout
- Add deployment automation for Streamlit Cloud or Azure App Service
- Introduce a project board and issue workflow for ongoing maintenance
